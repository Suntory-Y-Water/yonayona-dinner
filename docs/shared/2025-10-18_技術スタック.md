## 技術スタック

### フロントエンド
- React + Vite  
- Tailwind CSS (shadcn/ui)
- Google Maps JavaScript API（地図描画・ピン表示）  
- React Router(ルーティング)
- date-fns + date-fns-tz（営業時間・跨ぎ計算）

### バックエンド
- Hono（Google Places 実行基盤）  
- Cloudflare Workers（実行環境）  
- Cloudflare KV / Cache API（キャッシュ）  
- Google Places API（Nearby / Details）

### デプロイ
- フロントエンド、API：Cloudflare Workers

### セキュリティ
- APIキーはHono環境変数のみで保持  
- クライアントからGoogle APIへ直接アクセス禁止
